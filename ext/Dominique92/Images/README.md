# PhpBB 3.3 Resize images & slideshow

PhpBB3.3 extension

FUNCTIONS
=========
* Resize the images downloaded
```
download/file.php?id=4&s=1234
```
* Extract EXIF in phpbb-attachments.exif SQL table.field

* Display viewtopic as slideshow
